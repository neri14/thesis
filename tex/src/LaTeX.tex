% Wstawienie abstractu pracy
	%\input {abstract}
	
\abstractSH{
	W wyniku niniejszej pracy opracowany został inteligentny system sterowania ruchem drogowym, wykorzystujący do sterowania metody optymalizacji.
}

\abstractPL{
	Niniejsza praca jest próbą stworzenia inteligentnego systemu sterowania ruchem drogowym. Przygotowany został algorytm sterowania ruchem oparty o matematyczny model ruchu drogowego.\\
	Działanie algorytmu przebadane zostało na symulatorze używającym automatu komórkowego działającego w oparciu o zmodyfikowany model Nagela-Schreckenberga. Charakterystyka ruchu drogowego kontrolowanego przez opracowany algorytm sterowania została porównana z charakterystyką ruchu niekontrolowanego i kontrolowanego przez prosty program sygnalizacji.
}

\abstractEN{
	This work is an attempt to create intelligent traffic control system. Prepared algorithm is based on mathematical model of road traffic.\\
	Algorithm has been tested on traffic simulator using cellular automaton operating on the basis of modified Nagel-Schreckenberg model. Characteristic of road traffic controlled by prepared algorithm was compared to characteristics of not controlled traffic and traffic controlled by simple signaling program.
}
